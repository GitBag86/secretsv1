# Full Cycle Workflow

Trigger: new feature or app request

STEP 1 — Orchestrator
- break into tasks

STEP 2 — Architect
- design system

STEP 3 — Backend Agent
- implement API

STEP 4 — Frontend Agent
- implement UI

STEP 5 — DevOps Agent
- containerize system

STEP 6 — Security Agent
- review and patch risks

STEP 7 — Reviewer Agent
- final validation gate

LOOP RULE:
If Reviewer FAILS → return to relevant agent
